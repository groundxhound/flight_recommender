# Web Application (UI) - Travel Destination Recommender 
#
## In order to run the app, do the following:
#
# 1. Create a virtual environemnt inside the root directory of APP (/RecommenderApp).
#		- Open cmd prompt
# 		- Run: virtualenv <env_name> (make sure virtualenv is installed, if not: pip install virtualenv)
#		- It will create <env_name> directory in the root directory of app
# 2. Activate virtual environment: Go to /root_dir/<env_name>\Scripts\activate
# 3. Now install all dependencies: pip install -r requirements.txt
# 4. Now run the app: python app.py
# 5. Open a browser and log on to: http://localhost/ (port specification isn't needed here as app is running on port 80)
